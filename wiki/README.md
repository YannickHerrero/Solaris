# Solaris Wiki

Welcome to the official wiki for **Solaris**, a terminal-based idle game where you harness the power of the cosmos to generate infinite energy.

## Game Overview

Solaris is an incremental/idle game inspired by classics like Cookie Clicker. Starting with simple Solar Panels, you'll progress through 20 increasingly powerful energy producers, from Mining Drones to Big Bang Catalysts that create pocket universes.

## Key Features

### Energy Production
- **20 Unique Producers**: Each with 5x scaling in cost and production
- **Producer Upgrades**: 15 tiers per producer, each doubling production
- **Synergy System**: Adjacent producers boost each other's output

### Prestige System
- **Stellar Chips**: Earned by ascending, calculated from total energy
- **24 Prestige Upgrades**: Permanent bonuses that persist across runs
- **Multiple Progression Paths**: Production multipliers, starting bonuses, unlock shortcuts

### Quality of Life
- **Offline Progress**: Earn energy while away (up to 8 hours)
- **Auto-Save**: Automatic saving every 30 seconds
- **Boss Mode**: Quick-hide spreadsheet disguise
- **Flexible UI**: Adapts between single-column and two-column layouts
